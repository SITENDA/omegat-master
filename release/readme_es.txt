¿Qué es OmegaT?
===============

OmegaT es una herramienta multiplataforma libre para la traducción asistida por 
computadora (TAC) que está dotada de diversas funciones: coincidencias aproximadas, 
memoria de traducción, búsqueda por palabras clave, glosarios y aprovechamiento de 
traducciones previas en proyectos nuevos.

Prerrequisitos para instalar OmegaT
===================================

OmegaT requiere que tenga instalado en su sistema Java Runtime Environment (JRE) 
versión 1.8 o superior.

Recomendamos usar paquetes de OmegaT que incluyan JRE para evitarle la molestia 
de seleccionarlo, obtenerlo e instalarlo.



Instalar OmegaT (Windows)
===========================

Iniciar el programa de instalación.



Instalar OmegaT (Mac)
=======================

Descomprima el archivo .zip de OmegaT para obtener una carpeta que contiene
un archivo de documentación y la aplicación de OmegaT. Mueva la carpeta a una ubicación adecuada como
la carpeta Aplicaciones.



Instalar OmegaT (Linux)
=========================

Ponga el archivo en una carpeta y descomprímala. OmegaT ya está listo
para iniciarse.

Sin embargo, puede obtener una instalación más limpia y fácil de usar
mediante el programa de instalación (linux-install.sh). Para usar este programa, abra una
ventana de terminal (consola), cambie la carpeta a aquella que contiene
OmegaT.jar y el programa linux-install.sh y ejecute el programa con ./linux-install.sh.



Instalar OmegaT (Solaris, FreeBSD, etc.)
=========================================

Ponga el archivo en una carpeta y descomprímala. OmegaT ya está listo
para iniciarse.



Usar Java Web Start para instalar OmegaT (todas las plataformas)
===========================================================

Si ya tiene Java instalado en su sistema, una manera de instalar OmegaT
es usar Java Web Start.

Para ello descargue y ejecute el siguiente archivo:

  https://omegat.sourceforge.net/webstart/OmegaT.jnlp

Este instalará el entorno correcto para su equipo y la aplicación en sí misma
en la primera ejecución. En posteriores llamadas no es necesario estar
conectado a Internet.



Iniciar OmegaT (Windows)
==========================

Si durante la instalación creó un acceso directo en el
escritorio, haga doble clic en el acceso directo.

El instalador puede crear accesos directos para usted en el menú
de inicio, en el escritorio y en el área de inicio rápido. También puede arrastrar manualmente el archivo
OmegaT.exe al menú Inicio, al escritorio o al área de inicio rápido
para enlazarlo desde allí.

Si puede ver el archivo OmegaT pero no OmegaT.exe en el
Administrador de archivos (Windows Explorer), cambie la
configuración para que muestre la extensión de los archivos.



Iniciar OmegaT (Mac)
======================

Haga doble clic en la aplicación OmegaT.

Puede arrastrar la aplicación OmegaT al dock o a la barra de herramientas
de una ventana de Finder para poder iniciarla desde cualquier ubicación. También puede iniciarla
desde el campo de búsqueda
de Spotlight.



Iniciar OmegaT (Linux)
========================

Si usó el programa linux-install.sh, debería poder
iniciar OmegaT con:

    Alt+F2

y luego:

    omegat

Una manera más fácil de iniciar OmegaT es con
el programa Kaptain que está incluido (omegat.kaptn). Para usar este programa, primero debe instalar Kaptain. Entonces
podrá iniciar el programa de inicio en Kaptain con:

    Alt+F2

y luego:

    omegat.kaptn



Iniciar OmegaT desde la línea de comandos (todos los sistemas)
====================================================

El comando para iniciar OmegaT es:

    cd <directorio donde está el archivo OmegaT.jar>

    <nombre y ruta del archivo Java ejecutable> -jar OmegaT.jar

(El archivo ejecutable de Java es el archivo java en Linux y
java.exe en Windows.  Si
Java está instalado a nivel de sistema y está en la ruta de comandos, no tiene
que introducir la ruta completa.)
